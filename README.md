# 🩺 Diabetes Disease Prediction System

A Machine Learning-based web application that predicts whether a person is likely to have diabetes based on medical parameters.

---

## 🚀 Project Overview

This project uses a trained Machine Learning model to analyze health data and predict diabetes risk. It is deployed using **Streamlit** for an interactive user interface.

---

## 📊 Features

* ✔️ User-friendly web interface
* ✔️ Real-time prediction
* ✔️ Data visualization (graphs)
* ✔️ Scaled input data using StandardScaler
* ✔️ Trained ML model for high accuracy

---

## 🧠 Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Streamlit

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ankitkr756208-hash/Diabetes-Disease-Prediction-System.git
cd Diabetes-Disease-Prediction-System
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
streamlit run app.py
```

---

## 📈 Input Features

The model uses the following medical attributes:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

---

## 🧪 Model Used

* Logistic Regression / Random Forest (based on your notebook)
* Data scaled using **StandardScaler**

---

## 📷 Output

* Shows whether the person is:

  * ✅ Diabetic
  * ❌ Non-Diabetic
